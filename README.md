# Shared Agentics

A portable library of universal [Claude Code](https://claude.ai/code) skills for building agentic flows.

## Skills

| Skill | What it does |
|-------|-------------|
| **meta-prompt** | Creates, chains, and audits prompts and Claude Code skills optimized for Claude 4.6 |
| **meta-workflow** | Designs multi-agent pipelines with steps, gates, and context flow |
| **meta-agent** | Creates agent definitions (workers, orchestrators, reviewers) from task descriptions or workflows |
| **meta-hook** | Creates, audits, and debugs Claude Code hooks (lifecycle guardrails, context injection, validation) |
| **meta-architect** | Reviews project architecture, detects smells, generates diagrams, designs systems from scratch |
| **meta-research** | Decomposes vague questions into structured research plans with sub-questions, sources, and sufficiency criteria |
